Project: AI CV Chatbot with RAG (Retrieval-Augmented Generation)
Developed an intelligent chatbot system that answers questions about CV, skills, and projects using RAG technology.
The system uses open-source LLMs (Mistral-7B via Ollama), FAISS vector database for semantic search, and 
sentence-transformers for embeddings. Built a full-stack application with FastAPI backend and React frontend,
featuring chat history persistence with localStorage, modern UI design, and conversational AI capabilities.
The chatbot prevents hallucination by grounding all responses in the provided CV data and provides natural,
human-like conversations in both English and Turkish.
